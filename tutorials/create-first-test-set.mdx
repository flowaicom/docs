## What You'll Build

In this tutorial, you'll learn how to:
- Set up the Flow AI SDK
- Create a new project
- Upload your dataset
- Generate your first test set
- Validate and export the results

## Prerequisites

- Python 3.8 or higher
- Flow AI API key
- A sample dataset (CSV or JSON format)

## Step 1: Install and Configure the SDK

First, install the Flow AI SDK:

```bash
pip install flow-ai-sdk
```

Configure your API key:

```python
from flow_ai import FlowAI

# Initialize the client with your API key
client = FlowAI(api_key="your-api-key-here")

# Verify authentication
user_info = client.get_user_info()
print(f"Connected as: {user_info['name']}")
```

## Step 2: Create a New Project

```python
# Create a new project
project = client.create_project(
    name="My First Test Set Project",
    description="Learning how to generate test sets with Flow AI",
    settings={
        "quality_threshold": 0.8,
        "validation_required": True
    }
)

print(f"Project created with ID: {project['id']}")
```

## Step 3: Upload Your Dataset

```python
# Upload a dataset file
dataset = client.upload_file(
    project_id=project['id'],
    file_path="path/to/your/dataset.csv",
    file_type="csv"
)

print(f"Dataset uploaded: {dataset['id']}")
```

## Step 4: Configure Test Set Generation

```python
# Configure test set generation parameters
config = {
    "test_set_size": 100,
    "generation_method": "automated",
    "quality_settings": {
        "diversity_threshold": 0.7,
        "coverage_requirements": ["edge_cases", "typical_scenarios"]
    }
}

# Start test set generation
job = client.create_test_set(
    project_id=project['id'],
    dataset_id=dataset['id'],
    config=config
)

print(f"Test set generation started: {job['id']}")
```

## Step 5: Monitor Progress

```python
import time

# Monitor the generation progress
while True:
    status = client.get_job_status(job['id'])
    print(f"Status: {status['status']} - Progress: {status['progress']}%")
    
    if status['status'] in ['completed', 'failed']:
        break
    
    time.sleep(10)  # Check every 10 seconds
```

## Step 6: Review and Validate Results

```python
if status['status'] == 'completed':
    # Get the generated test set
    test_set = client.get_test_set(job['test_set_id'])
    
    # Review quality metrics
    metrics = client.get_test_set_metrics(test_set['id'])
    print(f"Quality score: {metrics['quality_score']}")
    print(f"Coverage: {metrics['coverage_percentage']}%")
    
    # Preview first few test cases
    preview = client.get_test_set_preview(test_set['id'], limit=5)
    for i, test_case in enumerate(preview['items']):
        print(f"Test case {i+1}: {test_case['input']} -> {test_case['expected_output']}")
```

## Step 7: Export Your Test Set

```python
# Export the test set in your preferred format
export_job = client.export_test_set(
    test_set_id=test_set['id'],
    format="json",
    destination="local"
)

# Download the exported file
client.download_export(export_job['id'], "my_first_test_set.json")
print("Test set exported successfully!")
```

## Complete Example

Here's the complete code for reference:

```python
from flow_ai import FlowAI
import time

# Initialize client
client = FlowAI(api_key="your-api-key-here")

# Create project
project = client.create_project(
    name="My First Test Set Project",
    description="Learning how to generate test sets",
    settings={"quality_threshold": 0.8}
)

# Upload dataset
dataset = client.upload_file(
    project_id=project['id'],
    file_path="dataset.csv",
    file_type="csv"
)

# Generate test set
job = client.create_test_set(
    project_id=project['id'],
    dataset_id=dataset['id'],
    config={
        "test_set_size": 100,
        "generation_method": "automated"
    }
)

# Monitor progress
while True:
    status = client.get_job_status(job['id'])
    if status['status'] == 'completed':
        break
    time.sleep(10)

# Export results
test_set = client.get_test_set(job['test_set_id'])
client.export_test_set(test_set['id'], format="json")
```

**Common Issues:**

1. **Authentication errors**: Verify your API key is correct
2. **File upload failures**: Check file format and size limits
3. **Generation timeouts**: Large datasets may take longer to process
