# Get Started with Flow AI

Get up and running with Flow AI. Create expert-validated test datasets for your AI agents with our simple SDK.

## 1. Create Your Account

Sign up for Flow AI and access your dashboard:

1. Go to [flow-ai.com/app](https://flow-ai.com/app)
2. Sign up for your account
3. Get Your API Key

<Frame>
  <img src="/images/dashboard-placeholder.png" alt="Flow AI Dashboard"/>
</Frame>


## 3. Install the SDK

```bash
pip install flowai_sdk
```

## 4. Set Up Your Environment

Create a `.env` file in your project root:

```bash
FLOWAI_API_KEY=your_api_key_here
```

## 5. Connect and Create Project

```python
import os
from dotenv import load_dotenv
from flowai_sdk import FlowAI

load_dotenv()

client = FlowAI()

# Login to get session token
login_response = client.authentication.sdk_login(
    http_headers={"Authorization": f"Bearer {os.getenv('FLOWAI_API_KEY')}"}
)

auth_headers = {"Authorization": f"Bearer {login_response.jwt}"}

# Create your first project
project = client.projects.create(
    name="My First Project",
    description="Getting started with Flow AI",
    http_headers=auth_headers
)

project_id = project.id
print(f"âœ… Connected and created project: {project_id}")
```

## 6. Create Your First Test Dataset

```python
# Upload your test data files
stages = client.stages.create_from_files(
    file_paths=["agent_rules.yaml"],
    source_type="AGENT_RULES", 
    project_id=project_id,
    http_headers=auth_headers
)

# Generate test cases and batch
job_response = client.jobs.trigger_pipeline_from_stages(
    project_id=project_id,
    stages=stages,
    http_headers=auth_headers
)
```

## Need Help?

**Common Issues:**
- **API key errors**: Make sure your key starts with `flowai_` and is in your `.env` file
- **File not found**: Check that your data files exist in the correct path
- **Project creation**: Ensure you have the necessary permissions in your account
